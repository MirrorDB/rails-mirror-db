Rails MirrorReflection

RubyGEM that reflect your DataBase models in a Rails project

View project on GitHub

MirrorReflection

Installation

Add the MirrorReflection to your application's Gemfile :

gem 'mirror_reflection'

And then execute :

$ bundle install

Or install it yourself as :

$ gem install mirror_reflection

Usage

  1. Install the Rails (3.2 or greater)
    • $ gem install rails
  2. Install the MirrorGEM

  3. Generate your rails project

    $ rails new app_name

  4. Access your app folder

    $ cd app_name

  5. Run the mirror installation

    $ rails g mirror_reflection:install

  6. Run the mirror reflect (it runs scaffolds, so lets while the mirror work for you )

        # DBMS = SGBD 
    rake mirror:reflect[dbms_name database_name username password host]

    Obs: if your DB connections are not default (localhost, root, no_password)



Congratilation! your new app mirrored is up.
